## 📌 What is GitHub?
GitHub is a web-based platform that helps developers host, manage, and track changes in their projects using **Git**. It provides features like:

✅ **Version Control**  
✅ **Collaboration & Pull Requests**  
✅ **CI/CD Integrations**  
✅ **Project Management Tools**  
✅ **Secure Code Hosting**
